Taskana_App — это современное приложение для работы со списком дел (todo-менеджер). Проект находится в активной разработке и является моей первой практической работой в рамках акселератора PREAX.

**✨ Текущий функционал**
На данный момент в приложении реализованы все базовые операции для комфортного управления задачами:
✅ Создание задачи — быстрое добавление новой записи
✏️ Редактирование — изменение существующих задач
🗑️ Удаление — ненужные задачи удаляются в один клик
✔️ Отметка о выполнении — удобный чекбокс для отслеживания прогресса
🔄 Сортировка — задачи можно сортировать по нескольким критериям:
    - По дате создания
    - По дате обновления
    - По алфавиту
    - По приоритету

**🎨 Интерфейс и дизайн**
- Темная тема — комфортная работа в любое время суток
- Адаптивный дизайн — приложение одинаково удобно на десктопе, планшете и мобильном телефоне
- Плавные анимации — приятные визуальные эффекты при взаимодействии

**🛠 Стек технологий**
Языки: HTML5, CSS3, JavaScript
Фреймворк: React
Сборка: Vite
Контроль версий: Git

**🚀 Планы по развитию**
Проект активно развивается. В ближайших планах:
👤 Добавление авторизации пользователей
📁 Создание личного кабинета
☁️ Синхронизация данных между устройствами

**Установка зависимостей** - npm install
**Запуск проекта в dev режиме** - npm run dev
**Сборка проекта в build версию** - npm run build
